## Qué hago
Facilito la comprensión visual de arquitecturas complejas y lógicas condicionales generando diagramas representativos en texto.

## Cuándo usarme
Usa esta habilidad al documentar la arquitectura de routing (enrutamiento de intents), el pipeline de RAG o cualquier flujo de LangGraph en los tutoriales.

## Reglas de ejecución
1. **Visualización obligatoria:** No uses solo texto para explicar un flujo de datos o un grafo.
2. **Uso de diagramas:** Inserta diagramas generados en sintaxis de bloque de código usando Mermaid.js.
3. **Claridad:** Etiqueta explícitamente las Entradas de usuario, Nodos de Decisión, LLMs invocados y Salidas estructuradas.
